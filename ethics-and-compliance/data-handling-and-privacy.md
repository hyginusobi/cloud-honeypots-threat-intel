# Data Handling and Privacy

## Data Types Collected
The project collected only technical security telemetry, including:
- Network connection metadata
- Authentication attempts
- Command execution logs within honeypots

No personal data was intentionally collected.

---

## Privacy Controls
- IP addresses treated as potentially sensitive
- Logs anonymised where required
- No enrichment with external identity data

---

## Data Storage
- Logs stored securely
- Access restricted to the researcher
- Retention limited to research needs

---

## Data Usage
Collected data was used exclusively for:
- Threat behaviour analysis
- Detection engineering
- Academic reporting

---

## Privacy Outcome
The project demonstrates responsible data stewardship aligned with privacy-by-design principles.
